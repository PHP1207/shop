BLOG 博客开发入门教程：前言

BLOG 博客教程 2017年5月11日 BLOG 4 评论 588 阅读

BLOG 是使用 Python 编写的一个开源 Web 框架，可以用来快速搭建一个高性能的网站。

BLOG makes it easier to build better Web apps more quickly and with less code.

BLOG 让你以更快的速度、更少的代码、更轻松的方式搭建更好的 Web 应用。

本教程将带你使用 BLOG 快速开发属于自己的 Blog 网站。

教程特点

免费、中文、零基础，完整的项目，基于最新版 BLOG 1.10 和 Python 3.5。

带你从零开始一步步开发属于自己的博客网站，帮助你以最快的速度掌握 BLOG 开发的技巧。

谁适合这个教程

本教程主要面向零基础的 BLOG 新人。

只需要一点点 Python 语言的基础，就可以顺利阅读本教程。

如果你已有一定的 BLOG 开发经验，也能从本教程中学到更多的 BLOG 开发技巧。
